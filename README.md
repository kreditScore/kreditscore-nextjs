# KreditScore Next.js Website

Modern, responsive loan and credit score website built with **Next.js 14**, **TypeScript**, **Tailwind CSS**, and **Framer Motion**.

## 🚀 Features

### ✅ Complete Sections Implemented:

1. **8 Loan Products** with icons and features:
   - Fresh Personal Loan
   - Pre-Approved Loan
   - Pay Credit Card Bill
   - Debt Consolidation
   - Salaried Over Draft
   - Small Apps Loan
   - Balance Transfer
   - Existing Loan Transfer

2. **4 Financial Tools**:
   - Company Category
   - Credit Score Checker
   - EMI Calculator
   - Part Calculator

3. **3-Step How It Works**:
   - Quick & easy online forms
   - Submit minimum documents
   - Funds sent fast

4. **6 Why Choose Us Points**:
   - Experienced Loan Advisors
   - Tech Friendly
   - One Loan Professional
   - Data Security
   - Convenient Contactability
   - Associations (40+ Banks & NBFCs)

### 🎨 Design Features:
- ✨ Smooth animations with Framer Motion
- 📱 Fully responsive (mobile, tablet, desktop)
- 🎭 Modern gradient backgrounds
- 🖱️ Hover effects and transitions
- 🎯 Lucide React icons
- 🌈 Tailwind CSS utility classes
- ⚡ AI-powered messaging
- 💬 WhatsApp floating button

## 📦 Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Fonts**: Poppins (Google Fonts)

## 🛠️ Installation

### Prerequisites:
- Node.js 18+ installed
- npm or yarn package manager

### Steps:

1. **Navigate to project directory:**
   ```bash
   cd c:\Users\Dell\Documents\BOOTCAMP\kreditscore-nextjs
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

   or with yarn:
   ```bash
   yarn install
   ```

3. **Run development server:**
   ```bash
   npm run dev
   ```

   or with yarn:
   ```bash
   yarn dev
   ```

4. **Open browser:**
   Navigate to [http://localhost:3000](http://localhost:3000)
